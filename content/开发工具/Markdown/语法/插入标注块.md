标注块（Callout）是 Obsidian 和 Quartz 都支持的特殊语法，用来**高亮提示信息**，渲染后会带有图标和背景色。

## 基本语法

```markdown
> [!类型]
> 标注内容写在这里
> 每一行前面都要加 `> `（大于号 + 空格）
```

## 常用类型

| 类型    | 写法               | 用途        |
| ----- | ---------------- | --------- |
| 📝 备注 | `> [!NOTE]`      | 补充说明、背景信息 |
| 💡 提示 | `> [!TIP]`       | 最佳实践、效率技巧 |
| ❗ 重要  | `> [!IMPORTANT]` | 必须知道的关键信息 |
| ⚠️ 警告 | `> [!WARNING]`   | 可能出问题的地方  |
| 🔴 危险 | `> [!CAUTION]`   | 高风险操作提醒   |
|       |                  |           |

## 示例与效果

**写法：**

```markdown
> [!NOTE]
> Frontmatter 是写给程序看的，正文是写给人看的。

> [!TIP]
> 按 `Ctrl + P` 可以快速打开命令面板。

> [!WARNING]
> 正文中不要再写 `# 标题`，否则会跟 Frontmatter 的 title 重复显示。
```

**效果：**

> [!NOTE]
> Frontmatter 是写给程序看的，正文是写给人看的。

## 自定义标题

默认标题是类型名（如 "Tip"），可以在类型后面加自定义标题：

```markdown
> [!TIP] 快捷键小技巧
> 按 `Ctrl + P` 可以快速打开命令面板。
```

> [!TIP] 快捷键小技巧
